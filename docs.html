How to create levels?

If you know programming, this will be easy for you to remember.

Basics

Hotkeys:

Ctrl + R Reload text style
Ctrl + O Open a file
Ctrl + S Save the file

Fields:

Every line contains an instruction. Every instruction has arguments.

Instructions "red", "orange", "yellow", "lime", "cyan"

These have 5 arguments. But first you need to provide the color and the appear time (in ms). Example:

red 1000

Then you need to set the position of the top left corner of the square. You can make it using two numbers from 0 to 100. Example:

red 1000 25 35

And the last - you need to set the size with two numbers. Examples:

red 1000 25 35 10 15 cyan 2505 95 0 5 48 yellow 1337 69 69 13 13

Instruction "text"

It's similar to the previous instructions, but you provide the text, set the position of the center and you don't provide size. Example:

text 1000 25 35 Text example. Cool, right?

Instruction "shakebg"

Shakes the background. You provide the appear time, strength, repeat times and interval between repeats. Example:

shakebg 3000 30 30 30

Instruction "blurbg"

Sets the blurriness of the background. You provide the appear time and blur strength. Example:

blurbg 3000 0

Instruction "flashbg"

Flashes the background. You provide only the appear time. Example:

shakebg 3000

"r"

"r" can replace the color, a position number or a scale number.
